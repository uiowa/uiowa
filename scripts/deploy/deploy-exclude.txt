# Supplemental exclude list for the deploy artifact.

# scripts/deploy/assemble.sh feeds this to rsync as --exclude-from AND has rsync
# honor .gitignore (--filter ':- .gitignore'). .gitignore is the source of truth
# for everything not in version control: secrets, local config, and the
# composer/yarn-managed trees (vendor, docroot/core, contrib, node_modules,
# compiled assets) that the build regenerates. This file only covers the gaps:
# tracked files we don't want at runtime, plus git metadata.
#
# rsync pattern semantics (not gitignore): a leading slash anchors to the build
# root; a pattern without a slash matches at any depth.

# are committable into the artifact rather than re-ignored.
